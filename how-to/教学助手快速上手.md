# 欢迎使用 Inno 教学助手 🎒

*适用版本 v0.4.1*

这是一份面向老师的图文上手指南。你**不需要任何技术背景**——打开网页,用大白话说清楚"几年级、什么学科、哪节课",它就帮你把**教案、题目、知识讲解、课件、课堂互动网页**做出来,并存成可直接用的文件。

Inno 有**两种用法**,这份文档按它们分成两章:

- **第一章 · 简单模式**——开箱即用。选一张助手卡片,直接出成品,适合日常备课。
- **第二章 · 普通模式**——完整功能。管理多个项目、给 AI 配长期记忆、记笔记、设定时任务、配置模型与飞书,适合深度使用与管理配置。

两种模式**随时切换、数据互通**,你可以先用简单模式上手,需要时再进普通模式。

> 💬 **读不下去也没关系——直接问它就行。** 这份文档里的任何操作、配置,你都可以**直接在对话框里用大白话跟助手说**,让它带你做。比如不知道怎么接飞书,就发一句:
>
> ```
> 我想把助手接到飞书,该怎么配?一步步教我
> ```
>
> 它会一步步告诉你点哪里、填什么。换句话说,**这份说明书本身,也可以"问"出来。**

> 😇 用的过程中有任何不顺手,欢迎反馈,我们会持续完善这份文档～

## 🤔 它能帮我做什么?

把它想成一位**随叫随到的备课搭子**:你说一节课,它出成品。

- 要上课 → 帮你写**教案**
- 要练习 → 帮你出一套**分层练习题**(还自动配答案解析)
- 要讲清楚 → 帮你把知识点**讲得通俗**,分基础/重点/难点/易错点
- 要上大屏 → 帮你做一个**课堂互动网页**(随堂答题、动画演示)
- 要课件 → 帮你生成可编辑的 **PPT**

而且产出都会**存成文件**,随时回看、下载、继续改。

## 🧭 进入「简单模式」,选一个助手

- 进页面后,如果**没看到一排教学卡片**,点输入框附近的**简单模式开关**切过去(再点一下可切回普通模式)。
- 简单模式的欢迎页会出现一排卡片:**「开箱即用 · 选一个开始」**。
- **点哪张卡,就进入哪个助手**;之后像聊天一样把需求说清楚即可。
- 它做出来的东西都会**自动存成文件**,放在这个助手的专属空间里,随时能看、能下载。

💡 **不想选助手?直接问也行**——简单模式下,**不点任何卡片**,直接在底部输入框里发消息,就进入通用问答。备课思路、解释知识点、帮你改一段话都行;**需不需要把结果存成文件,看你怎么说**——只想聊聊就纯聊,让它"做一份""导出来",它也会照办。

## 📂 做好的东西在哪、怎么拿

- 每个助手生成的文件,都放在它**自己的空间**里,界面上能浏览、下载。
- **教案、讲解稿、题目**是 `.md` 文档（Markdown 格式),用记事本 / Word / 任意文本编辑器都能打开,也可直接把内容复制走。
- **教学网页**是一个 `.html` 网页文件,下载后双击用浏览器打开,再投到教室大屏。
- **PPT** 是可编辑的 `.pptx` 演示文稿。
- 同一个助手反复用,会用同一个空间,历次成果归档在一起,方便回头查。

下载 `.html` 文件后:① 在文件管理器里双击,系统会用浏览器打开;② 投影时可按 `Ctrl +`(Windows)或 `⌘ +`(Mac)整体放大页面,适配不同屏幕距离。

## ⚙️ 配置模型

打开**设置 → 模型 / 服务提供方**,新增一个模型来源:

1. **提供方 ID(Provider)**:起个标识,如 `anthropic`、`deepseek`。
2. **接口地址(Base URL)**:服务商给的 API 地址,如 Anthropic 官方 `https://api.anthropic.com`、DeepSeek `https://api.deepseek.com`。
3. **接口类型(API)**:
   - Anthropic / Claude 官方 → 选 `anthropic-messages`
   - OpenAI、DeepSeek、通义等"OpenAI 兼容"服务 → 选 `openai-completions`
4. **API Key**:把申请到的密钥粘进来(当密码保管,别外泄)。
5. **模型 ID**:服务商规定的模型名,如 `claude-sonnet-4-6`、`deepseek-chat`。
6. **模型名称**:给自己看的名字,随意。

填好**保存**,再到**界面顶部的模型选择器**里选为当前模型(选择会被记住)。

![设置页的 服务提供方 / 模型表单](assets/shot-14.png)

> 😅 **发消息后一直转圈、没回复?** 九成是 API Key 或接口类型填错了:Anthropic / Claude 官方选 `anthropic-messages`,OpenAI / DeepSeek 选 `openai-completions`,填错会静默失败。

## 📱 接入飞书

把助手接到飞书,**用手机就能随时对话**,相当于把教学助手装进飞书。

> ✅ 本系统用飞书的**长连接(WebSocket)**方式收发消息——**不需要公网服务器,也不用填回调地址**,只要 App ID 和 App Secret。

![飞书里和 Inno 机器人对话](assets/shot-15.png)

**第一步:在飞书开放平台建应用**

1. 打开飞书开放平台(open.feishu.cn)→ 开发者后台 → **创建企业自建应用**。
2. 进应用 →「凭证与基础信息」,记下 **App ID** 和 **App Secret**。
3. 在「应用能力」里**启用「机器人」**。
4. 「权限管理」开通**消息相关权限**:接收用户发给机器人的消息、发送消息(以及上传图片 / 文件以便回传)。
5. 「事件订阅」选**长连接(使用 SDK / WebSocket)方式**(不要选填 URL 的那种),添加事件 **「接收消息 im.message.receive_v1」**。
6. 按平台提示**创建版本并发布**,让权限生效。

![飞书开放平台·事件订阅:长连接方式 + 接收消息事件](assets/shot-16.png)

**第二步:在 Inno 网页里填上**

1. 打开**设置 → 消息渠道 → 飞书**。
2. 填 **App ID**、**App Secret**,打开**飞书开关**。
3. (可选)**仅私聊**:默认开,只回应一对一私聊;**允许的用户**:留空 = 不限,或填用户 ID 白名单(每行一个)。
4. **保存**;重启服务后,飞书会自动建立长连接。

![设置 → 消息渠道 → 飞书,填好 App ID / Secret 并开启开关](assets/shot-17.png)

**第三步:开用**——在飞书里搜到并**私聊这个机器人**,直接发消息,就和在网页里一样对话。

## 🖥️ 界面长什么样(三栏)

普通模式是经典的三栏布局:

- **左侧栏**——你的**工作区**和**历史对话**,在这里新建、切换、搜索、归档。

- **右侧面板**——**多个标签页**:预览文件、笔记本、学习者画像、定时任务、技能、设置。

## 🗂️ 左侧栏:工作区与对话

**工作区(Workspace)** 就像一个"项目文件夹",里面装着围绕同一主题的多次对话和它们产出的文件。左侧栏的工作区从上到下分成几组:

- **你的项目**——你自己新建的工作区,可重命名、可删除。
- **渠道工作区**——从飞书 / 微信 / 命令行 过来的对话,系统**自动建立、自动归类**(不能改名或删除)。
- **临时**——随手聊的临时空间,同样由系统自动管理。
- **已归档**——你**归档过的对话**会收到这里,固定在最底部、默认折叠,随时可恢复(注意:归档的是单条对话,不是整个工作区)。

常用操作:

- **新建对话 / 新建**(`+` 按钮)——在选中的工作区里开一段新对话。
- **重命名 / 删除**——把鼠标移到**你自己建的工作区**上,会出现对应小图标(渠道 / 临时工作区由系统托管,没有这些操作)。
- **归档对话**——把鼠标移到**单条对话**上,点归档图标即可收起;在"已归档"组里可恢复。
- **搜索框**——按名字快速找历史对话。
- **渠道筛选**——只看来自某个渠道的对话,渠道有 **Web / 飞书 / 微信 / CLI / Job** 五种。

每条对话会显示名称、时间,以及**来源渠道**的小标记。这里的 **Job** 指**定时任务自动触发**产生的对话(见后面「定时任务」),它只是一种来源标记,既不是工作区分组、也不等于"已归档"。

## 💬 对话区:不止于打字

中间的输入框除了打字,还能传东西给 AI:

- **文字**——`Enter` 发送,`Shift + Enter` 换行。
- **📎 传文件**——点回形针图标上传资料(教材、PDF、表格等),AI 会据此回答,并可归档进知识库。
- **🖱️ 拖拽上传**——也可以把电脑里的文件**直接拖到右侧「预览」面板**松手,即上传到当前工作区(拖拽的落点是右侧文件面板,不是输入框)。
- **🖼️ 传图片**——点图片图标选图,或直接 `Ctrl / ⌘ + V` **粘贴截图**;可以发题目照片、板书、图表让它识别。
- **停止 / 重试**——生成过程中可随时点「停止」;万一某条失败,会出现「重试」。

## 🧠 让 AI 拥有记忆:L1 / L2 / L3

普通模式和简单模式最大的不同,是 AI 能**记住东西**。三层记忆可在 **设置 → 记忆** 里分别开关:

- **L1 · 学习者画像**——记住"**使用你的人是谁**"(不限学生还是老师):学习目标、哪些概念掌握了、常犯什么错、喜欢什么讲法 / 学法。开启后每轮对话自动带上,实现**因人而异的个性化**。
- **L2 · 知识库(笔记本)**——把对话里的资料归档成可检索的知识库,下次能随时调取。
- **L3 · 跨对话记忆**——跨越不同对话**自动回忆**相关内容,不用每次从头交代背景。

> 简单模式下这三层是统一关闭的——所以那里看不到笔记本、画像这些标签页。想用记忆功能,就在普通模式里开。

## ⏰ 定时任务

让助手按计划自动跑,适合"周期性的事"交给它:

- **任务类型**——内置每日复盘 / 每周总结 / 图谱更新 / 画像反思 / 间隔复习 / 推送提醒等几种;也可以选「自定义提示」,**用一段话让它定时做你描述的任何事**。
- **频率**——每天 / 每周 / 每月 / 工作日 / 仅一次 等都能点选,还能用**自定义 Cron** 精确到你想要的任意时间。
- 每个任务可启用 / 停用、立即运行、编辑、删除,并显示上次 / 下次运行时间。

## ✨ 技能

## 🙋 常见问题

**Q:简单模式和普通模式,我该用哪个?**

日常备课、要现成成品 → **简单模式**;想管理多个项目、给 AI 配记忆 / 画像、记笔记、设定时任务、改模型或接飞书 → **普通模式**。两者随时切换,数据互通。

---

**Q:普通模式会不会太复杂?**

不用全用。核心还是中间的对话框,右侧那些标签页**按需点开**即可。先当成"能记笔记、有记忆的简单模式"用就行。

---

**Q:没看到那排教学卡片?**

先确认在「简单模式」(切换开关在输入框附近 / 左侧栏);普通模式不显示这些卡片。若已经在简单模式却还看不到卡片(比如正停在某段历史对话里),**点一下左侧栏底部的「新建对话」**,就会回到带卡片的欢迎页。

---

**Q:它问得太细 / 答不到点上?**

开头一次把"年级 / 学科 / 课题 / 这节课要讲什么"说清楚,给得越具体、产出越贴合。

---

**Q:想改它做出来的东西?**

直接在对话里说,比如"把第 3 题换成填空""教案的导入再生动点",它会改并重新存档。

---

**Q:想换个助手?**

点左侧栏**底部的「新建对话」**,就回到简单模式欢迎页重新选卡片;各助手的文件和历史互不干扰。
