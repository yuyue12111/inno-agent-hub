# K12 数学核心素养 · 规范基座

这是一个**被引用的基座 skill**：学情分析、教学目标、教学活动、教学评价等阶段 skill 都以它为准，
统一「素养受控词、规范表述、标签写法、量规语言」，避免各处口径不一。依据《义务教育数学课程标准（2022 年版）》。

> 其他 skill 引用方式：读取 `../k12-core-competency/SKILL.md`，取用下方受控词与表述。

## 一、核心素养 = 「三会」

| 代号 | 三会 | 含义 |
|---|---|---|
| 数学眼光 | 会用数学的眼光**观察**现实世界 | 从现实中抽象出数学对象与结构 |
| 数学思维 | 会用数学的思维**思考**现实世界 | 用逻辑与运算严谨推演 |
| 数学语言 | 会用数学的语言**表达**现实世界 | 用符号、图表、模型刻画并交流 |

## 二、初中阶段 9 个主要表现（`competency` 受控词）

**取值必须落在这 9 个词内**，保证与学情 CSV、教学目标、评价量规完全一致：

| 三会归类 | 核心素养表现（受控词） |
|---|---|
| 数学眼光 | `抽象能力`、`几何直观`、`空间观念`、`创新意识` |
| 数学思维 | `运算能力`、`推理能力` |
| 数学语言 | `数据观念`、`模型观念`、`应用意识` |

> 小学阶段还含「数感、量感、符号意识」等 11 个表现；初中数学以上表 9 个为主。跨学段使用时按学段调整。

## 三、四基 / 四能（目标载体）

- **四基**：基础知识、基本技能、基本思想、基本活动经验。
- **四能**：发现问题、提出问题、分析问题、解决问题。
- 教学目标应显式标注四基/四能落点；素养是「里子」，四基四能是「抓手」。

## 四、教学评一致性（逆向设计）

先定**目标** → 再定**评价证据/任务** → 再设计**教学活动**，三者对齐：

```
目标  →  评价证据/量规  →  教学活动
  └────────── 对齐检查（教-学-评一致）──────────┘
```

## 五、素养标签写法（供教学目标使用）

每条教学目标末尾用括号标注对应素养表现，格式：**（学科-素养表现）**。
示例："……能正确求出临界点，准确率达 80% 以上。**（数学-运算能力）**"。
一条目标可带 1–2 个素养标签，避免堆砌。

## 六、分等级量规语言约定（供教学评价使用）

统一用 **等级零 → 等级四** 五档，描述必须**可观察、可勾选**，且体现能力递进：

| 等级 | 语言基调（示例锚点） |
|---|---|
| 等级零 | 无法独立开始，需教师全程一对一示范 |
| 等级一 | 在支架/提示下能模仿完成部分，准确率低、需多次提示 |
| 等级二 | 能在模板支持下独立完成主干，达到基本标准（常设为「合格线」） |
| 等级三 | 能独立、规范完成并自我校验（**常设为预期达成等级**） |
| 等级四 | 能迁移到新情境，并能向同伴讲解、发现并纠正他人错误 |

> 写量规时把上面的「基调」锤定到**具体课题的可观察行为**上，不要停留在通用措辞。

## 七、使用清单（给引用它的阶段 skill）

- [ ] `competency` 一律取自第二节的 9 个受控词。
- [ ] 目标标签用第五节格式。
- [ ] 量规用第六节的五档语言，并落到具体行为。
- [ ] 目标—评价—活动三者对齐（第四节）。
