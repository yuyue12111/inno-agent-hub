# 教学内容设计 · 从目标到可讲授的课件

把教学目标落成一节课的可讲授内容，产出一份**课堂课件（PPT）**。
**实际生成 PPT 走 `pptx` skill**（本 skill 负责内容结构与取向，pptx skill 负责排版出片）。

## 一、先定内容取向

出片前先确认内容更偏哪种取向：
- 情境/问题驱动为主
- 概念讲解 + 例题精讲为主
- 探究发现式为主

## 二、课件结构（建议页序）

1. 封面：课题、课型、课时。
2. 学习目标页：引用阶段二目标（可精简为学生能听懂的版本）。
3. 情境导入：用阶段二的核心问题/教学情境引入，制造认知需求。
4. 核心概念：本课要建立的概念，配图示，少堆公式。
5. 例题/探究：**例子优先**——典型例题分步讲解，或探究任务分步引导；覆盖学情薄弱点。
6. 变式/巩固：一两道变式题。
7. 课堂小结：方法链/知识结构图。

## 三、质量准则

- 内容与阶段二目标、阶段一薄弱项一致。
- 例子优先、可视化，不堆纯理论/纯公式。
- 每页信息量克制，适合课堂投影与录屏。
- 默认套用品牌母版配色（见工作区 `pptx母版/`，pptx skill 内已内置）。

## 四、自检清单

- [ ] 已确认内容取向。
- [ ] 覆盖学习目标页 + 情境导入 + 核心概念 + 例题/探究 + 小结。
- [ ] 例题针对学情薄弱点。
- [ ] 通过 pptx skill 生成，套用母版配色。
