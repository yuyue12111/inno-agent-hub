# 随堂练习构建

当用户要为某节课出题时,按本技能产出两份卷子。

## 产出形式(两份 Markdown 文件)

- **学生卷**:只含题目与作答区,不含答案。
- **教师卷**:同一套题,逐题附「对应知识点 / 学习目标 + 参考答案 + 详细解析」。

## 题型与分层

- **题型按学科适配**:选择、填空、判断、简答、实操 / 计算等。
- **三个层级**,每层标注清楚:
  - 基础巩固
  - 能力提升
  - 拓展提高
- **题量适配**当堂检测,不贪多;每题都对应到本节课的具体知识点。

## 每题结构(教师卷)

- 题号 + 题型 + 层级标签
- 对应知识点 / 学习目标
- 题干(及选项)
- 参考答案
- 详细解析(讲思路、点易错)

## 工作流程

1. 确认年级、学科、本节课核心知识点(可取用工作区里已有的教案 / 讲解稿)。
2. 先列出题目分布(各题型 / 各层级各几题),确认后再展开。
3. 产出学生卷 + 教师卷两个 Markdown 文件,写入当前工作区,并告知文件位置。
