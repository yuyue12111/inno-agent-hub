# Skill Library

可复用的 Skill 包收集。每个 `.md` 文件可以直接通过工作区文件树的 **✦ 按钮**上传到 Inno Agent。

## Skill 列表

| 文件 | 名称 | 适用场景 |
|---|---|---|
| [card-maker.md](./card-maker.md) | 词汇卡片生成器 | 英语学习，把生词整理成 Anki CSV |

## 使用方法

1. 下载对应的 `.md` 文件
2. 在 Inno Agent Web UI 打开目标工作区的「预览」标签页
3. 点击文件树工具栏的 **✦** 按钮，选择该文件上传
4. 新建一个绑定该工作区的会话即可使用
