# tavily-search

通过 [Tavily](https://tavily.com) 为 Inno Agent 提供实时网络搜索能力。

## 使用步骤

**1. 申请 API Key**

前往 [https://www.tavily.com/](https://www.tavily.com/) 注册并获取 API Key。

免费额度：每月 **1000 次**搜索。

**2. 填入 Key**

打开本目录的 `SKILL.md`，将 `YOUR_TAVILY_API_KEY` 替换为你的真实 Key。

**3. 上传**

在工作区「预览」页，点击文件树工具栏 **✦** 按钮，选择修改后的 `SKILL.md` 上传。
