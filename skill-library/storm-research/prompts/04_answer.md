# Step 4 — 检索作答（抗幻觉）(Grounded Answer)

> 作用：强制每句话落在检索回来的证据上，宁可拒答也不编造——多视角的信息差靠这一步才是真的。

## 内核

你是善用信息的专家，已经收集了相关信息，现在用这些信息组织回答。

## 提示词模板

```
讨论的主题：{TOPIC}
问题：{QUESTION}
已收集的信息（来自 Step 3 的真实检索）：
{RETRIEVED_INFO 带 [n] 编号}

要求：
- 让回答尽可能翔实，但确保**每一句话都有上面收集的信息支撑**。
- 在句末用 [n] 标注该句依据的来源编号。
- 尽量使用不同来源，不要只靠一条。
- 如果收集的信息与问题不直接相关，给出基于现有信息最相关的回答。
- 如果无法形成有依据的回答，明确写："基于现有信息无法回答这个问题。"并说明缺口或局限。
- 绝对不要用模型记忆/常识来填补检索空白。
```

## 为什么这样设计

这是抗幻觉的一等公民设计。宁可拒答，不可编造。同时注意截断未完成的带引用句子，避免输出半截的伪引用——若发现某句引用对不上来源表，删掉该句。
